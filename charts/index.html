TWHDB

The Women's Health Database

A searchable, filterable overview of longitudinal women's-health cohort studies: what each cohort collects, how it classifies procedure/data validity, and where in the world it's based. Data is refreshed manually from the lab's source spreadsheet whenever the team runs an update.

Tip

Click any row to see that cohort's full record. Rows are color-coded by Procedure Separation Type — see the legend below the search box. Click a column header to sort.

How to use this

Build one or more conditions below — for example, "N is greater than 1000" or "Hot flashes item: equals Yes". Then choose whether a cohort needs to match all of your conditions (AND) or any of them (OR) using the dropdown. The table updates instantly as you add, edit, or remove conditions, so it's a quick way to answer questions like "which cohorts are prospective and include a hysterectomy question?" without digging through the raw spreadsheet.
Tip

Hover a marker to preview a cohort; click it for full details. Marker size reflects sample size (N), and marker color follows the same Procedure Separation Type coding used on the Cohort Summary tab.
